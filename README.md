# Be Kind Social Agency Hub

## Meta API permissions

For analytics sync and monthly report generation, Meta tokens should include:

- `pages_read_engagement`
- `pages_show_list`
- `instagram_basic`
- `instagram_manage_insights`
- `instagram_content_publish` (future use)

## Local frontend API URL

Add `VITE_API_URL` to `artifacts/agency-portal/.env.local` when running frontend locally.
