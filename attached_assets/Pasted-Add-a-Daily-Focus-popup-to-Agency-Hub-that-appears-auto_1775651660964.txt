Add a Daily Focus popup to Agency Hub that appears automatically 
when any team member opens the portal each day.
Do not modify anything that already exists.

---

OVERVIEW:
A smart daily productivity popup that shows the team member 
their most important task for the day, one at a time, 
ordered by a priority matrix based on urgency and importance.
The goal is to help the team focus on what truly matters 
each day without feeling overwhelmed.

---

1. TRIGGER BEHAVIOR

The popup appears automatically:
- Once per day per user, on first login of the day
- If the user closes it and reopens the portal within the same day: 
  does NOT show again (tracked via localStorage + Supabase)
- Exception: if new urgent tasks were added since last seen, 
  show again with only the new urgent ones
- User can manually reopen it anytime via:
  * Top bar button: "Today's Focus" with a target icon
  * Keyboard shortcut: Cmd+K (or Ctrl+K on Windows)

---

2. PRIORITY MATRIX (Eisenhower Matrix)

Tasks are automatically scored and sorted using this logic:

QUADRANT 1 — DO FIRST (Urgente + Importante):
- Due today or overdue
- Priority set to "urgente" or "alta"
- Assigned to the current user
- Shown with red accent, label "🔴 Da fare subito"

QUADRANT 2 — SCHEDULE (Importante, non urgente):
- Due within the next 7 days
- Priority "alta" or "media"
- Assigned to current user
- Shown with purple accent, label "🟣 Pianifica oggi"

QUADRANT 3 — DELEGATE (Urgente, non importante):
- Due today or tomorrow
- Priority "bassa"
- Could be reassigned
- Shown with amber accent, label "🟡 Valuta se delegare"

QUADRANT 4 — ELIMINATE (Non urgente, non importante):
- Due far in future
- Priority "bassa"
- No hard deadline
- Shown with gray accent, label "⚪ Bassa priorità"

Scoring algorithm:
- Base score by priority: urgente=100, alta=75, media=50, bassa=25
- Days until due: overdue=+50, today=+40, tomorrow=+30, 
  within 3 days=+20, within 7 days=+10, beyond=0
- Has client dependency: +15
- Has blockers (other tasks waiting on this): +10
- Estimated hours (shorter tasks get slight boost for quick wins): 
  under 1h=+5
- Final score determines the order shown

3. POPUP UI

SIZE AND POSITION:
- Centered modal, 560px wide, auto height
- Backdrop: dark semi-transparent overlay
- Cannot be dismissed by clicking outside (must use buttons)
- Smooth entrance animation: scale up from center

HEADER:
- Left: greeting based on time of day:
  * Before 12:00 — "Buongiorno, [Name] ☀️"
  * 12:00–17:00 — "Buon pomeriggio, [Name] 👋"  
  * After 17:00 — "Buonasera, [Name] 🌙"
- Right: "X of Y tasks" counter (e.g. "1 di 5")
- Below greeting: date in Italian format 
  "Lunedì, 7 Aprile 2026"

PROGRESS BAR:
- Thin colored bar below header showing position in task list
- Fills as user moves through tasks
- Color matches current quadrant (red/purple/amber/gray)

TASK CARD (main content — one task at a time):

┌──────────────────────────────────────────┐
│  [QUADRANT BADGE]          [CLIENT NAME] │
│                                          │
│  [TASK TITLE — large, 20px bold]         │
│                                          │
│  [PROJECT NAME]  •  [PLATFORM/CATEGORY]  │
│                                          │
│  [DESCRIPTION — max 2 lines, truncated]  │
│                                          │
│  ┌────────────┬────────────┬───────────┐ │
│  │ ⏰ Scadenza │ ⏱ Stimato  │ 👤 Client │ │
│  │ [date]     │ [Xh]       │ [name]   │ │
│  └────────────┴────────────┴───────────┘ │
│                                          │
│  [PRIORITY SCORE BAR]                    │
│  "Priorità: [score]/100"                 │
│                                          │
│  [QUADRANT EXPLANATION — 1 sentence]     │
│  e.g. "Questa task è in scadenza oggi    │
│  e richiede la tua attenzione immediata" │
└──────────────────────────────────────────┘

QUADRANT EXPLANATION sentences:
- Q1: "In scadenza oggi — completala prima di tutto il resto"
- Q2: "Importante per il progetto — pianifica quando farla oggi"
- Q3: "Urgente ma a bassa priorità — valuta se delegarla al team"
- Q4: "Non urgente — tienila in mente ma non bloccarti ora"

ACTION BUTTONS (below the card):

Primary row:
[✓ Inizia ora]  [→ Prossima]  [✗ Salta]

- "Inizia ora": marks task as "in_corso" in database, 
  closes popup and navigates directly to the task detail page
- "Prossima": moves to next task without action, 
  remembers user saw this one
- "Salta": removes task from today's focus list for this user 
  (not deleted, just hidden from today's popup)

Secondary row (smaller, text buttons):
[Delega al team]  [Sposta a domani]  [Segna come completata]

- "Delega al team": opens a small inline panel to reassign 
  to another team member with a note
- "Sposta a domani": updates due_date +1 day with a note "Posticipata da [user]"
- "Segna come completata": marks done, shows micro-celebration animation,
  moves to next task automatically

4. NAVIGATION BETWEEN TASKS

SWIPE GESTURES (touch/trackpad):
- Swipe left → next task
- Swipe right → previous task

KEYBOARD SHORTCUTS:
- Arrow Right / N → next task
- Arrow Left / P → previous task
- Enter → start task now
- S → skip
- C → mark complete
- D → delegate

TASK COUNTER DOTS:
- Row of dots below action buttons
- One dot per task (max 10 dots shown, rest as "+X more")
- Current task dot is filled, others are empty
- Clicking a dot jumps to that task

- "Ricorda: inizia sempre dalla prima della lista!"
