Add a complete Time Tracker system to Agency Hub, integrated with 
tasks and clients. Do not modify anything that already exists.

---

OVERVIEW:
A professional time tracking tool that automatically starts when 
a team member clicks "Inizia ora" from the Daily Focus popup or 
manually from any task. Every minute worked is logged, attributed 
to a client and project, giving the agency a clear picture of 
how much time is invested per client each month.

---

1. HOW IT STARTS

AUTOMATIC START:
- When user clicks "Inizia ora" in the Daily Focus popup
  → timer starts immediately for that task
- When user clicks "Inizia" button on any task detail page
  → timer starts for that task

MANUAL START:
- Floating timer button in the top bar (clock icon)
- Click → dropdown to select:
  * Client (required)
  * Project (filtered by selected client)
  * Task (filtered by selected project, optional)
  * Description (free text, optional)
  * Click "Avvia timer"

QUICK START from any page:
- On every task card and task detail page add a 
  ▶ Play button that starts the timer instantly
- On every project page add "Track time" button
- On every client page add "Track time" button

---

2. ACTIVE TIMER WIDGET

When a timer is running show a persistent widget in the top bar:
- Pulsing red dot (recording indicator)
- Client name + task name (truncated)
- Live counter: HH:MM:SS counting up in real time
- Pause button (⏸)
- Stop button (⏹)
- The widget is always visible on every page while timer runs
- Only one timer can run at a time per user
- If user tries to start a second timer:
  Show prompt: "Hai già un timer attivo per [task].
  Vuoi fermarlo e iniziarne uno nuovo?"
  Options: "Ferma e inizia nuovo" / "Annulla"

PAUSE BEHAVIOR:
- Paused timer shows gray dot instead of red
- Counter freezes
- Resume button (▶) to continue
- Pause reason optional: 
  "Pausa pranzo / Riunione / Interruzione / Altro"
- Paused time is NOT counted in the total

---

ACTIVITY TYPES (customizable):
- Strategia e pianificazione
- Creazione contenuti
- Grafica e design
- Copywriting
- Gestione campagne ADV
- Reportistica
- Riunione / Call con cliente
- Riunione interna
- Revisioni e feedback
- Amministrazione
- Altro

BILLABLE TOGGLE:
- Default ON for all client work
- Default OFF for: internal meetings, admin, training
- Billable hours tracked separately for invoicing purposes

---

4. MANUAL TIME ENTRY

Users can add time entries manually (for time worked offline,
forgotten to track, etc.):

From the Time Tracker section → "+ Aggiungi tempo manuale"

Fields:
- Client (required)
- Project (required)
- Task (optional)
- Description (required)
- Activity type
- Date (default: today)
- Start time
- End time
- OR: Duration (hours + minutes)
- Billable: yes/no
- Note: "Inserimento manuale" label always shown on manual entries

- Rows: Client / Project / Task combinations

12. REMINDERS AND ALERTS

- If user has no active timer for more than 2 hours during work hours:
  Gentle nudge notification: "Stai lavorando? Non dimenticare 
  di avviare il timer ⏱"
  (work hours defined in user settings, default 9am–6pm)

- End of day reminder (customizable time, default 6pm):
  "Hai tracciato [Xh] oggi. Ricordati di fermare il timer 
  se hai finito di lavorare."

- Weekly reminder every Friday at 5pm:
  "Hai inviato il timesheet questa settimana?"

- Monthly alert on last working day:
  "Fine mese! Controlla il riepilogo ore per la fatturazione."
