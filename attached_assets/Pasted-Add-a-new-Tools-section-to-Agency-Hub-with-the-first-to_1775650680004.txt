Add a new "Tools" section to Agency Hub with the first tool being 
a fully functional Editorial Plan (Piano Editoriale) builder.
Do not modify anything that already exists.

---

OVERVIEW:
A monthly editorial plan builder that lets the team create, customize 
and export a professional content calendar for each client.
The plan is exportable as a PDF to send to the client for approval 
before content is produced and published.

Fields:
- Select Client (dropdown from existing clients database)
- Month (dropdown: January–December)

- Platforms to include (multi-select checkboxes):

  * Instagram Reels
  * Facebook Feed
  * Facebook Stories
  * LinkedIn
  * TikTok
  * YouTube Shorts
- Content package (affects how many slots are pre-generated):
  * Base — 4 feed posts/month
  * Standard — 8 feed posts/month

4. PLAN BUILDER (main editor)

LAYOUT:
- Left panel (280px): Content slots list
- Right panel: Selected slot editor
- Top bar: client name, month/year, status badge, 
  action buttons (Save / Preview PDF / Send for Review)

LEFT PANEL — CONTENT SLOTS:
Content slots are organized by platform in collapsible sections.
Each platform section shows:
- Platform icon + name
- Total slots for that platform
- Progress: [X completed / Y total]

Each content slot card shows:
- Slot number (es. Post #1, Reel #1)
- Content category (see categories below)
- Planned publish date
- Status dot: empty / in progress / ready / approved
- Thumbnail if image/video has been attached

Clicking a slot opens it in the RIGHT PANEL editor.

---

5. CONTENT SLOT EDITOR (right panel)

Each slot has these fields:

BASIC INFO:
- Platform (pre-set from setup, can be changed)
- Content Type: Post / Reel / Story / Carousel / Video
- Content Category (fully customizable — see section 6)
- Planned publish date (date picker)
- Planned publish time (time picker)
- Slot status: Da creare / In lavorazione / Pronto / Approvato

CONTENT DETAILS:
- Post title / working title (internal reference)
- Caption (rich text — supports emoji, line breaks, hashtags)
  * Character counter (Instagram max: 2200)
  * AI assist button: "Generate caption ideas" 
    → opens AI chat with context pre-filled
- Hashtags (tag input field, shows count — Instagram max: 30)
  * Button: "Suggest hashtags" → AI suggests relevant hashtags
- Call to Action (dropdown): 
  Shop Now / Learn More / Visit Profile / Send Message / 
  Save this post / Tag a friend / Custom text
- Link in bio (if applicable)

VISUAL REFERENCE:
- Upload reference image or video thumbnail
- Or paste a reference URL (Canva link, Google Drive link, etc.)
- Or add a text description of the visual: 
  "Photo of product on white background, lifestyle mood"
- Visual format reminder shown: 
  Feed: 1080x1080 or 1080x1350 | Reel: 1080x1920 | Story: 1080x1920

NOTES:
- Internal notes for the creative team (not shown in PDF)
- Client notes (shown in PDF as "Note per il cliente")

reorder, change color and icon for each):

Default list:
- Prodotto / Servizio (purple) — showcase products or services
- Educativo / Tips (blue) — educational content, how-to, tips

- Intrattenimento (orange) — entertaining, trending, viral content
- Carosello Informativo (cyan) — informative carousel / slides
- Collaborazione (violet) — influencer, partner content

Each category has:

- Color (color picker)
- Icon (select from icon library)
- Description (tooltip shown in builder)

CATEGORY MANAGER:
Accessible from Tools → Settings → Categorie Contenuto

- Edit existing

- Reorder via drag and drop
- Categories are global (apply to all clients) OR 
  can be set as client-specific overrides

10. DUPLICATE & TEMPLATE SYSTEM

DUPLICATE PLAN:
- "Duplicate" button on any existing plan
- Creates a copy for the same or different client/month
- All content slots copied, dates shifted to new month automatically
- Status reset to "Bozza"

SAVE AS TEMPLATE:
- "Save as Template" option on any completed plan
- Template stored with a name (es. "Standard 8 post — Fashion Brand")
- When creating a new plan, option to "Start from template"
- Templates available: custom saved + system defaults

SYSTEM TEMPLATE DEFAULTS:
- Standard Social Media (8 posts/month balanced mix)
- E-commerce Focus (12 posts — product heavy)
- B2B LinkedIn Focus (8 posts — educational heavy)
- Launch Campaign (12 posts — awareness + conversion mix)

---

11. COLLABORATION FEATURES

INTERNAL REVIEW FLOW:
1. Creator clicks "Invia per revisione interna"
2. Team members assigned to that client get notified
3. They can comment on individual slots (comment icon on each card)
4. Once approved internally → status changes to "Approvato"
5. "Invia PDF al Cliente" button unlocks

SLOT COMMENTS:
- Each content slot has a comments thread (internal only, not in PDF)
- Team can discuss specific posts: "Change the caption tone" / "Need new visual"
- Comments resolved with a checkmark

CHANGE HISTORY:
- Every edit to a slot is logged with: user, timestamp, what changed
- Accessible via "History" icon on each slot
