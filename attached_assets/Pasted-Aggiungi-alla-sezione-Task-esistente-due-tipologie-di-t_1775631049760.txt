TIPOLOGIA 2 — TASK AVANZATA CON CHECKLIST
Quando si crea una task avanzata, l'utente sceglie una categoria e si carica 
automaticamente la checklist predefinita relativa. Tutte le voci sono spuntabili 
singolarmente. L'avanzamento della task è calcolato in % sulle voci completate.

---

CATEGORIA: "Piano Editoriale Mensile"
L'utente seleziona il mese (Gennaio...Dicembre) e il pacchetto contenuti:
  - Pacchetto Base: 4 contenuti/mese feed
  - Pacchetto Standard: 8 contenuti/mese feed
  - Pacchetto Premium: 12 contenuti/mese feed

Checklist generata automaticamente in base al pacchetto:
[ ] PED [mese selezionato] - Piano editoriale creato
[ ] Template grafici creati:
    [ ] Template Carosello
    [ ] Template Storia
    [ ] Template Post IG
[ ] Contenuti Foto/Video creati ([N] su [TOT])
[ ] Contenuti grafici creati ([N] su [TOT])
[ ] Programmazione completata
[ ] Pubblicazioni verificate
[ ] Approvazione cliente ricevuta

La N viene calcolata automaticamente: 
- 4 contenuti = 4 post feed + 8 stories + 2 reel
- 8 contenuti = 8 post feed + 16 stories + 4 reel  
- 12 contenuti = 12 post feed + 24 stories + 6 reel

UI E FUNZIONALITÀ:

1. CREAZIONE TASK
Nella modale di creazione task aggiungere un toggle:
"Tipo task: [ Semplice ] [ Avanzata con checklist ]"
Se si sceglie Avanzata: mostra dropdown categoria, 
poi appare la checklist precompilata modificabile prima del salvataggio.

2. VISTA TASK AVANZATA
Nel dettaglio della task mostrare:
- Barra di avanzamento % (voci spuntate / totale voci)
- Lista checklist con checkbox interattive
- Contatore "[X/Y] completati"
- Ogni sotto-voce può essere spuntata da qualsiasi membro del team
- Le voci spuntate si aggiornano in tempo reale (Supabase Realtime)

3. NELLA BOARD KANBAN
Le task avanzate mostrano:
- Badge con la categoria (es. "Piano Editoriale")
- Mini barra di progresso sotto il titolo
- Contatore voci completate

4. NELLA SCHEDA CLIENTE/PROGETTO
Aggiungi una sezione "Avanzamento operativo" che mostra:
- Per ogni task avanzata attiva: nome, categoria, % completamento
- Checklist Onboarding Cliente con avanzamento globale
- Totale contenuti da produrre nel mese (in base al pacchetto scelto)
  con contatore: "Contenuti creati: [X] su [N]"

5. DATABASE
Aggiorna la tabella task aggiungendo:
- tipo: 'semplice' | 'avanzata'
- categoria: string
- checklist_json: array di oggetti {id, testo, completato, gruppo}
- pacchetto_contenuti: '4' | '8' | '12' (solo per categoria Piano Editoriale)
- mese_riferimento: string (solo per Piano Editoriale)

Usa dati mock realistici per mostrare subito esempi di entrambe le tipologie 
nella board kanban.
