IMPORTANT API NOTE:

The media_url from Instagram Graph API is a direct CDN link.
To display it in the app:
- Fetch the URL server-side (Next.js API route) to avoid CORS issues
- Cache the image URL in Supabase for 24 hours to avoid repeated API calls
- For videos, only thumbnail_url is needed (not the video file itself)
- Always include the permalink so the client can click through 
  to the original post
